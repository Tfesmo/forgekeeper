# Forgekeeper

An interactive terminal-based AI chat client for software development.

Forgekeeper is a terminal CLI with a focus on aggressive context management.  It prunes context when that grows too large, while preserving your agents.md instructions and guiding your agent to keep on task.  Long term session memory is preserved by guiding the agent to take notes in a configured vector database (e.g. Chroma, Qdrant).
